# Feature Hierarchy and Priorities

## Tier 1: Must-have MVP

### Customer
- Home / landing
- nearby live map
- truck detail page
- open now filter
- cuisine filter
- directions CTA
- last updated visibility

### Owner
- auth and role selection
- create / claim truck
- truck profile editor
- owner dashboard
- go live / go offline
- live location update
- menu manager
- schedule manager

### Shared
- static / dynamic / schedule data separation
- freshness timestamps
- stale degradation
- basic analytics event capture

---

## Tier 2: Strong differentiators

- favorites / follows
- favorites page
- booking request form
- owner booking inbox
- owner analytics page
- customer confirmation trust layer
- recurring schedule templates
- owner reminders

---

## Tier 3: Later expansion

- reviews
- loyalty / rewards
- advanced event tools
- promoted placements
- native mobile app
- push notifications
- team accounts
- ordering integrations
- route planning
- deeper admin tools

---

## Priority rules

Build features that improve:
1. trust
2. discovery speed
3. owner update ease
4. owner ROI
5. retention

Avoid features that mainly add complexity without improving those five.
